# Security Policy

## Supported versions

Only the active groundwork branch and latest `main` receive security fixes during initial development.

## Reporting a vulnerability

Email the maintainers through the contact method listed on the Supernova Horizon organization profile. Do not open public issues for undisclosed vulnerabilities.

## Secrets

Never commit `.env` files, API tokens, AWS credentials, client lead data, or production account identifiers to this public repository.
