# Demo Landscaping

Multi-page fictional landscaping demonstration site for the Supernova Site Platform.

- Fixture: `evergreenGroveFixture` from `@supernova/vertical-landscaping`
- Dev server: `pnpm dev` on port `4321`
- Pages: home, services, service areas, about, projects, FAQ, contact, quote, privacy, and 404

This app is not a real business and must not include fake testimonials, ratings, or revenue claims.
